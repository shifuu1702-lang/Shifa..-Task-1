Data Cleaning Project (Excel)
Overview

I cleaned a raw dataset using Excel to remove errors and make the data ready for analysis.

Steps Done

Removed missing values using filters

Removed duplicate rows (Data → Remove Duplicates)

Standardized text (TRIM, PROPER, UPPER/LOWER)

Fixed date format to dd-mm-yyyy

Renamed column headers to clean and consistent names

Corrected data types (numbers, dates, text)

Files

raw_dataset.xlsx – original file

cleaned_dataset.xlsx – final cleaned file

Learning

Practiced Excel data cleaning: handling nulls, duplicates, formatting, and data preparation.
